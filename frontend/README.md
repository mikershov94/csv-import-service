## CI/CD notes for this repository

- Frontend CI workflow: `.github/workflows/frontend-ci.yml`
- Frontend CD workflow: `.github/workflows/frontend-cd.yml`
- Docker image: `ghcr.io/<owner>/csv-import-service-frontend`

### Required scripts

- `npm run typecheck`
- `npm run lint`
- `npm run build`

### CD behavior

- CD triggers on Git tags matching `frontend-v*` and can also run manually.
- The tag commit must be contained in `main`.
- The job publishes a Docker image to GHCR and uses GitHub `frontend-production` environment.

### Environment setup

- GitHub: create Environment `frontend-production` and add environment secrets as needed when deployment steps are introduced later.
